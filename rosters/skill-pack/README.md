# Rosters — Skill pack

Solo **formaciones** listas para torneos con **Skill Points** (ver [source/tiers/skill-pack.md](../../source/tiers/skill-pack.md)). No hay índices por equipo en esta carpeta: cada `.md` es una alineación.

## Convención de nombres

`[equipo]-[x]P-[y]S-[z]A[-variante].md`

- **P:** habilidades del pack como **primaria** (+20k TV típico c/u en desglose).
- **S:** como **secundaria** (+40k TV c/u).
- **A:** jugadores con **apilamiento** (≥2 habilidades del pack en la misma ficha); debe cuadrar con el tier del torneo.
- **`-variante`:** solo si hace falta desambiguar (p. ej. dos listas con el mismo x/y/z).

Tabla resumida con TV y nombre de build: [rosters/README.md](../README.md#rosters-actuales) (sección «Skill pack»).
