# Unión Élfica — Torneos con skill pack

Guía de builds para torneos donde se asignan **Skill Points** antes del torneo. Unión Élfica es **Tier 2** → **7 Puntos**, **máx. 1 secundaria**, **sin apilamiento** (ver [skill-pack.md](../../source/tiers/skill-pack.md)). Siempre revisar el pack del torneo.

Con skill pack el equipo se vuelve muy peligroso: puedes crear **Blodge Blitzers**, **Catchers sackers** y **Línea defensiva**.

---

## Roster 1 – "Meta estándar"

**Jugadores**

| Posición | Cantidad |
|----------|----------|
| Elfo Blitzer | 2 |
| Elfo Catcher | 2 |
| Elfo Lanzador | 1 |
| Elfo Línea | 6 |
| Rerolls | 3 |
| Apotecario | Sí |

**Skills (ej. 6 primarias + 1 secundaria)**

| Posición | Habilidades |
|----------|-------------|
| Blitzers | Esquivar ×2 (Dodge) |
| Catchers | Forcejear (Wrestle), Placar (Block) |
| Línea | Patada (Kick), Placaje Defensivo (Tackle) |
| Lanzador | Líder (Leader) |

**Por qué funciona**
- Blitzers Blodge + Echarse a un Lado (ya de roster) son muy difíciles de quitar.
- Catcher con Forcejear es excelente sacker.

---

## Roster 2 – "Anti bash"

**Skills**

| Posición | Habilidades |
|----------|-------------|
| Blitzers | Esquivar, Guardia (secundaria) |
| Catchers | Forcejear, Esquivar |
| Línea | Patada, Forcejear |
| Lanzador | Líder |

**Estrategia**
- Pantallas profundas.
- Robo de balón con Catcher.

---

## Roster 3 – "Ultra agresivo"

Para torneos con **8 skills**.

**Skills**

| Posición | Habilidades |
|----------|-------------|
| Blitzers | Esquivar ×2 |
| Catchers | Placar (Block), Forcejear |
| Línea | Patada, Placaje Defensivo, Forcejear |
| Lanzador | Líder |

**Ventaja**
- Equipo muy completo: scorer, sacker, defensa sólida.

---

## Habilidades clave (orden competitivo)

| Orden | Habilidad | Uso |
|-------|-----------|-----|
| 1º | **Esquivar** (Blitzers) | Blodge. |
| 2º | **Forcejear** (Catchers o Línea) | Sackers. |
| 3º | **Patada** | Presión desde turno 1. |
| 4º | **Placaje Defensivo** | Cazar ball carriers. |
| 5º | **Guardia** | Si el pack permite secundaria. |
