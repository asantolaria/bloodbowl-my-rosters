# Nobleza Imperial — Torneos con skill pack

Guía de builds para torneos donde se asignan **Skill Points** antes del torneo. Nobleza Imperial es **Tier 2** → **7 Puntos**, **máx. 1 secundaria**, **sin apilamiento** (ver [skill-pack.md](../../source/tiers/skill-pack.md)). Siempre revisar el pack del torneo.

Con skill pack el equipo mejora mucho si añades **Guardia** y **Esquivar**: control defensivo y portador fiable.

---

## Roster 1 – "Meta estándar"

**Jugadores**

| Posición | Cantidad |
|----------|----------|
| Bodyguard | 4 |
| Noble Blitzer | 2 |
| Retainer Línea | 5 |
| Imperial Thrower | 1 |
| Ogre | 1 |
| Rerolls | 3 |

**Skills (ej. 6 primarias + 1 secundaria)**

| Posición | Habilidades |
|----------|-------------|
| Bodyguards | Guardia ×2 |
| Blitzers | Esquivar, Placaje Defensivo (Tackle) |
| Retainer | Patada (Kick) |
| Ogre | Guardia |
| **Secundaria** | Golpe Mortífero (Blitzer) |

**Por qué funciona**
- Guardia en Bodyguards hace el equipo muy difícil de mover.
- Blitzer con Esquivar = portador o anotador fiable.

---

## Roster 2 – "Anti-elfos"

Muy popular contra equipos ágiles.

**Skills**

| Posición | Habilidades |
|----------|-------------|
| Bodyguards | Guardia ×2 |
| Blitzers | Placaje Defensivo, Esquivar |
| Retainers | Patada, Forcejear (Wrestle) |
| Ogre | Guardia |

**Estrategia**
- Controlar el campo.
- Cazar portadores del balón ágiles.

---

## Roster 3 – "Control brutal"

Para torneos con **8 skills** (o pack más generoso).

**Skills**

| Posición | Habilidades |
|----------|-------------|
| Bodyguards | Guardia ×2, Golpe Mortífero |
| Blitzers | Esquivar, Placaje Defensivo |
| Retainers | Patada, Forcejear |
| Ogre | Guardia |

**Ventaja**
- Muy fuerte contra equipos bash.

---

## Habilidades clave (orden real en torneo)

| Orden | Habilidad | Uso |
|-------|-----------|-----|
| 1º | **Guardia** (Bodyguards) | Control de espacio. |
| 2º | **Esquivar** (Blitzers) | Portador/anotador. |
| 3º | **Placaje Defensivo** | Cazar ball carriers. |
| 4º | **Patada** | Presionar desde turno 1. |
| 5º | **Forcejear** | Sackers. |
