# Rosters iniciales (1.000k)

Rosters de inicio a valoración 1.000k, según builds recomendados (p. ej. FUMBBL BB25). Siguen la **plantilla** ([../source/plantilla-roster.md](../source/plantilla-roster.md)) y la convención de nombre **[nombre del equipo] - [raza] - [TV]** en slug.

Ejemplo: `inicio-1000k-orcos-negros-1000k.md` = nombre "Inicio 1000k", raza Orcos Negros, TV 1.000k.

Índice completo de rosters: [rosters/README.md](../README.md) (tabla de rosters actuales y guías limite-valoracion / skill-pack).
