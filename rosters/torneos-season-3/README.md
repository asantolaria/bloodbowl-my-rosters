# Rosters — Torneos Season 3

Plantillas de referencia para **torneos genéricos** en **Blood Bowl 3ª temporada / BB2025** (sin el marco Skill Gold / Flowing del EuroBowl). Presupuestos de ejemplo: **~1.000k–1.100k** (Unión Élfica), **~1.060k–1.100k** (No Muertos), **~1.000k–1.060k** (Skavens: 2 RR+fans / 3 RR / 2 RR+apo / 5.º Lineman), **~1.100k+** (Orcos Negros con skills de torneo), **~1.100k–1.135k** (Elegidos del Caos: Troll / Ogro / Minotauro), **~1.150k–1.155k** (Nobleza Imperial: 1 o 2 Throwers), **~1.150k–1.200k** (Hombres Lagarto), etc.

**No sustituyen** al reglamento de tu evento: rerolls, apotecario, fans y paquetes de habilidades pueden variar.

**Datos de lista:** [`source/teams/`](../../source/teams/) (Nuffle Zone 2025).

---

## Convención de archivos

`torneo-s3-[slug-equipo]-[TV]k.md`

- **slug-equipo:** como en `source/teams/` (ej. `hombres-lagarto`).
