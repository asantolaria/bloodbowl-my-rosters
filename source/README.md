# Source — Datos Blood Bowl

Datos de referencia según [NuffleZone.com](https://nufflezone.com/) (Blood Bowl 3ª Temporada, 11-2025).

## Índice unificado

**[index.md](source/index.md)** — Navegación de equipos, habilidades, jugadores estrella y tablas de juego en un solo archivo.

## Carpetas (cada una con README)

| Carpeta | Contenido |
|---------|-----------|
| [**teams/**](source/teams/) | Un archivo por equipo: roster 2025, costes, estadísticas, habilidades, Pri/Sec. Ver [README](source/teams/README.md). |
| [**habilidades/**](source/habilidades/) | Una ficha por categoría de habilidades (Agilidad, Fuerza, General, etc.). Ver [README](source/habilidades/README.md). |
| [**jugadores-estrella/**](source/jugadores-estrella/) | Una ficha por jugador estrella: coste, estadísticas, equipos. Ver [README](source/jugadores-estrella/README.md). |
| [**tiers/**](source/tiers/) | Oro por tier, Skill pack y tabla Tier \| Equipos (torneos NAF/Matched Play). Ver [README](source/tiers/README.md). |
| [**tablas/**](source/tablas/) | Tablas de partida: heridas, clima, patada inicial, plegarias, experiencia/SPP. Ver [README](source/tablas/README.md). |
| [**images/**](source/images/) | Imágenes de equipos (logos). Ver [README](source/images/README.md). |

## Otros archivos

## Convenciones

- **Coste:** en miles (65k = 65.000 monedas).
- **CTD:** cantidad máxima por tipo de jugador (0-X o 0-1 para Big Guys).
- **Rerolls y Apotecario:** ver ficha de cada equipo en `teams/`.
