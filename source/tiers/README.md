# Tiers — Blood Bowl 3ª Temporada

Referencia para rosters de **torneo**: **EuroBowl 2026** (Skill Gold), **oro por tier** NAF genérico, y documentación **legada** del antiguo sistema de puntos de habilidad. El agente debe priorizar el **pack publicado por el evento**; si es #euro26, usar **eurobowl-2026.md**.

## Uso por el Roster Builder

1. Identificar el **formato del torneo**: si es **EuroBowl 2026 / #euro26**, seguir **eurobowl-2026.md** (Team Budget + Skill Gold + Flowing Funds). Si el reglamento cita **Skill Points** por tier GW, usar **skill-pack.md**. En otro caso, **oro-presupuesto.md** + reglamento del evento.
2. Obtener el **tier del equipo** desde la tabla Tier \| Equipos del pack que aplique.
3. No superar topes de presupuesto de equipo ni de oro de habilidades que indique el evento.
4. Para builds ya escritos en el repo: **`rosters/eurobowl-2026/`** (#euro26), **`rosters/iniciales/`** y demás `.md` en **`rosters/`** (ver índice).
